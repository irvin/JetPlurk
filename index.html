JetPlurk α

Surfing Plurk with Jetpack

Install

  1. Install Jetpack Prototype extension, than restart firefox.
  2. Click "Install..." button on the top-right side of this page.
  3. On the next page, scroll down and click "I know what I'm doing. Install it!",
    you can select "Auto-update this feature" to enable auto-update.
  4. Then you can see install successed page,
    click about:jetpack link to set your plurk account.
